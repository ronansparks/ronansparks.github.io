数组

只出现一次的数字

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

说明：

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

示例 1:

输入: [2,2,1]
输出: 1

示例 2:

输入: [4,1,2,1,2]
输出: 4

野猫君实现：

耗时24 ms，战胜95.83%的 Swift 提交记录

class Solution {
    func singleNumber(_ nums: [Int]) -> Int {
        if nums.count == 0 { return 0 }

        var single = 0
        for num in nums {
            single ^= num
        }

        return single
    }
}

最佳实现：

耗时 20 ms

class Solution {
    func singleNumber(_ nums: [Int]) -> Int {
        var num = nums.first!
        for i in 1..<nums.count {
            num = num ^ nums[i]
        }
        return num
    }
}

解析：

我首先进行了数组边界检测，而他直接使用var num = nums.first!强制解析，当传入空数组时，会崩溃。

因为他事先把数组第一个值赋值给了变量，他的遍历比我少遍历一个，耗时更短？

使用他的实现，并加入我的数组边界检测，最终结果为 16ms，竟然超越了他，😂随机误差？

使用位运算符中的按位异或运算符 ^ 比较两个数，返回一个数。用二进制表示，如果两个数的同一位相同，则该位输出0，即两个相同的数，二进制相同，最终会抵消为0，剩下的则为那个出现了一次的元素。

旋转数组

给定一个数组，将数组中的元素向右移动 k 个位置，其中 k 是非负数。

示例 1:

输入: [1,2,3,4,5,6,7] 和 k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右旋转 1 步: [7,1,2,3,4,5,6]
向右旋转 2 步: [6,7,1,2,3,4,5]
向右旋转 3 步: [5,6,7,1,2,3,4]

示例 2:

输入: [-1,-100,3,99] 和 k = 2
输出: [3,99,-1,-100]
解释: 
向右旋转 1 步: [99,-1,-100,3]
向右旋转 2 步: [3,99,-1,-100]

说明:

  • 尽可能想出更多的解决方案，至少有三种不同的方法可以解决这个问题。
  • 要求使用空间复杂度为 O(1) 的原地算法。

野猫君实现：

耗时28 ms，战胜68.70%的 Swift 提交记录

class Solution {
    func rotate(_ nums: inout [Int], _ k: Int) {
        if nums.count == 0 || k % nums.count == 0 {
            return  
        }

        var tempArray = nums
        for i in 0..<nums.count {
            tempArray[(i + k) % nums.count] = nums[i]
        }
        nums = tempArray
    }
}

最佳实现：

耗时20 ms

两个数组的交集 II

给定两个数组，写一个方法来计算它们的交集。

例如:
给定 nums1 = [1, 2, 2, 1], nums2 = [2, 2], 返回 [2, 2].

注意：

  • 输出结果中每个元素出现的次数，应与元素在两个数组中出现的次数一致。
  • 我们可以不考虑输出结果的顺序。

跟进:

  • 如果给定的数组已经排好序呢？你将如何优化你的算法？
  • 如果 nums1 的大小比 nums2 小很多，哪种方法更优？
  • 如果nums2的元素存储在磁盘上，内存是有限的，你不能一次加载所有的元素到内存中，你该怎么办？

野猫君实现：

耗时144 ms， 战胜15.38%的 Swift 提交记录

class Solution {
    func intersect(_ nums1: [Int], _ nums2: [Int]) -> [Int] {
        if nums1.count == 0 || nums2.count == 0 {
            return []
        }

        var result: [Int] = []
        var temp1 = nums1

        for num in nums2 {
            if let index = temp1.index(of: num)  {
                result.append(num)
                temp1.remove(at: index)
            }
        }

        return result
    }
}

最佳实现：

耗时20 ms

class Solution {
    func intersect(_ nums1: [Int], _ nums2: [Int]) -> [Int] {
    var result = [Int]()
    var dict = Dictionary<Int,Int>()
    if nums1.count == 0 || nums2.count == 0 {
        return result
    }
    for i in 0..<nums1.count {
        let keyStr = nums1[i]
        if dict.keys.contains(keyStr) {
            dict[keyStr] = dict[keyStr]! + 1
        }else{
            dict[keyStr] = 1
        }
    }
    for i in 0..<nums2.count {
        let keyStr2 = nums2[i]
        if dict.keys.contains(keyStr2) && dict[keyStr2] != 0 {
            result.append(keyStr2)
            dict[keyStr2] = dict[keyStr2]! - 1
        }
    }
    return result
    }
}

解析：

他创建一个字典，把数组1的值作为key，对应出现次数为value。如数组1位[1,2,1,2]，数组2位[2, 2]，那么新的字典为[1: 2, 2: 2]；再去遍历数组2，以数组2的值作为key去查询字典，如果存在，就把数组2的值添加到结果，并设置对应的字典value减一。

跟进：

如果已经是排好序的，就从头同时开始遍历两个数组。

class Solution {
   func intersect(_ nums1: [Int], _ nums2: [Int]) -> [Int] {
        var result = [Int]()
        var n1 = 0
        var n2 = 0
        while n1 < nums1.count && n2 < nums2.count {
            if nums1[n1] < nums2[n2]{
                n1 += 1
            }
            else {
                if nums1[n1] > nums2[n2]{
                    n2 += 1
                }
                else {
                    result.append(nums1[n1])
                    n1 += 1
                    n2 += 1
                }
            }
        }
        return result
    }
}

如果其中一个比另一个小很多，使用数组遍历的方法比更优，只需遍历最小的数量即可。

两数之和

给定一个整数数组和一个目标值，找出数组中和为目标值的两个数。

你可以假设每个输入只对应一种答案，且同样的元素不能被重复利用。

示例:

给定 nums = [2, 7, 11, 15], target = 9

因为 nums[0] + nums[1] = 2 + 7 = 9
所以返回 [0, 1]

野猫君实现：

耗时28 ms，战胜81.09%的 Swift 提交记录

class Solution {
    func twoSum(_ nums: [Int], _ target: Int) -> [Int] {
        var result: [Int] = []
        var dict: [Int: Int] = [:]

        for (index, num) in nums.enumerated() {
            let difference = target - num
            if dict[difference] == num {
                let pairedIndex = nums.index(of: difference)!
                result.append(pairedIndex)
                result.append(index)
            }
            else {
                dict[num] = target - num
            }
        }

        return result
    }
}

最佳实现：

耗时16 ms

class Solution {
    func twoSum(_ nums: [Int], _ target: Int) -> [Int] {
        var dic = [Int: Int]()
        for i in 0..<nums.count {
            if let a = dic[target - nums[i]] {
                if i != a {
                    return [i,a]
                }
            }else{
                dic.updateValue(i, forKey: nums[i])
            }
        }
        return []
    }
}

解析：

在前面看到了别人的哈希表实现，这里也首先想到利用字典，将数组的值作为key，将目标值减去该值的结果作为字典的value，检查数组的遍历值是否和字典里值相等。

他的实现可以少几个声明变量，使用的普通 for 循环，字典中以数组值作为key，数组索引作为value。

从排序数组中删除重复项

给定一个排序数组，你需要在原地删除重复出现的元素，使得每个元素只出现一次，返回移除后数组的新长度。

不要使用额外的数组空间，你必须在原地修改输入数组并在使用 O(1) 额外空间的条件下完成。

示例 1:

给定数组 nums = [1,1,2], 

函数应该返回新的长度 2, 并且原数组 nums 的前两个元素被修改为 1, 2。 

你不需要考虑数组中超出新长度后面的元素。

示例 2:

给定 nums = [0,0,1,1,1,2,2,3,3,4],

函数应该返回新的长度 5, 并且原数组 nums 的前五个元素被修改为 0, 1, 2, 3, 4。

你不需要考虑数组中超出新长度后面的元素。

说明:

为什么返回数值是整数，但输出的答案是数组呢?

请注意，输入数组是以“引用”方式传递的，这意味着在函数里修改输入数组对于调用者是可见的。

你可以想象内部操作如下:

// nums 是以“引用”方式传递的。也就是说，不对实参做任何拷贝
int len = removeDuplicates(nums);

// 在函数里修改输入数组对于调用者是可见的。
// 根据你的函数返回的长度, 它会打印出数组中该长度范围内的所有元素。
for (int i = 0; i < len; i++) {
    print(nums[i]);
}

野猫君实现：

耗时24 ms，战胜100%的 Swift 提交记录

class Solution {
    func removeDuplicates(_ nums: inout [Int]) -> Int {
        if nums.count <= 1 { return nums.count }

        var indicator = 0
        for index in 0..<nums.count {
            if nums[index] != nums[indicator] {
                indicator += 1
                nums[indicator] = nums[index]
            }
        }
        return indicator + 1
    }
}

解析：

首先判断处理数组个数，定义indicator记录不重复元素的位置，遍历数组，如果遍历到数值不相等，则把indicator前进一位。因为indicator 是从0开始，所以数组个数还要加一。

买卖股票的最佳时机 II

给定一个数组，它的第 i 个元素是一支给定股票第 i 天的价格。

设计一个算法来计算你所能获取的最大利润。你可以尽可能地完成更多的交易（多次买卖一支股票）。

注意：你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。

示例 1:

输入: [7,1,5,3,6,4]
输出: 7
解释: 在第 2 天（股票价格 = 1）的时候买入，在第 3 天（股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5-1 = 4 。
     随后，在第 4 天（股票价格 = 3）的时候买入，在第 5 天（股票价格 = 6）的时候卖出, 这笔交易所能获得利润 = 6-3 = 3 。

示例 2:

输入: [1,2,3,4,5]
输出: 4
解释: 在第 1 天（股票价格 = 1）的时候买入，在第 5 天 （股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5-1 = 4 。
     注意你不能在第 1 天和第 2 天接连购买股票，之后再将它们卖出。
     因为这样属于同时参与了多笔交易，你必须在再次购买前出售掉之前的股票。

示例 3:

输入: [7,6,4,3,1]
输出: 0
解释: 在这种情况下, 没有交易完成, 所以最大利润为 0。

野猫君实现：

耗时16 ms，战胜100%的 Swift 提交记录

class Solution {
    func maxProfit(_ prices: [Int]) -> Int {
        if prices.count == 0 { return 0 }

        var sum = 0
        var minPrice = prices[0]
        for i in 1..<prices.count {
            if prices[i] < minPrice {
                minPrice = prices[i]
            }
            else {
                sum += prices[i] - minPrice
                minPrice = prices[i]
            }
        }

        return sum
    }
}

解析：

使用贪心算法，做出当前最好的选择，追求局部最优解。设置最低价格为第一天价格，从第二天开始遍历，如果价格比第一天低，则把它设为最低价；第三天价格比第二天价格高，那么利润就是第三天价格减去第二天的价格，再把最低价设为第三天价格。

存在重复

给定一个整数数组，判断是否存在重复元素。

如果任何值在数组中出现至少两次，函数返回 true。如果数组中每个元素都不相同，则返回 false。

示例 1:

输入: [1,2,3,1]
输出: true

示例 2:

输入: [1,2,3,4]
输出: false

示例 3:

输入: [1,1,1,3,3,4,3,2,4,2]
输出: true

野猫君实现：

耗时76 ms，战胜45%的 Swift 提交记录

class Solution {
    func containsDuplicate(_ nums: [Int]) -> Bool {
        if nums.count <= 1 { return false }
        let sorted = nums.sorted()

        for i in 1..<sorted.count {
            if sorted[i] == sorted[i - 1] {
                return true
            }
        }
        return false
    }
}

最佳实现：

耗时48 ms

class Solution {
    func containsDuplicate(_ nums: [Int]) -> Bool {
        if nums.count <= 1 { return false}
        var checkNums: Set<Int> = [nums.first!]
        for i in 1..<nums.count {
            if !checkNums.insert(nums[i]).inserted {
                return true
            }
        }
        return false
    }
}

解析：

第一反应就是排序、遍历，性能不好。他使用了集合现成方法，因为集合里没有重复的数。还可以直接比较集合的个数是否与数组个数相等，return Set(nums).count != nums.count。

加一

给定一个非负整数组成的非空数组，在该数的基础上加一，返回一个新的数组。

最高位数字存放在数组的首位， 数组中每个元素只存储一个数字。

你可以假设除了整数 0 之外，这个整数不会以零开头。

示例 1:

输入: [1,2,3]
输出: [1,2,4]
解释: 输入数组表示数字 123。

示例 2:

输入: [4,3,2,1]
输出: [4,3,2,2]
解释: 输入数组表示数字 4321。

野猫君实现：

耗时20 ms，战胜41.05%的 Swift 提交记录

class Solution {
    func plusOne(_ digits: [Int]) -> [Int] {
        if digits.count == 0 { return [] }

        var carry = 1
        var result = digits
        for i in (0..<result.count).reversed() {
            result[i] = (digits[i] + carry) % 10
            carry = (digits[i] + carry) / 10
        }

        if carry == 1 {
            result.insert(carry, at: 0)
        }
        return result
    }
}

最佳实现：

耗时12 ms

class Solution {
    // 方法二的改进
    func plusOne(_ digits: [Int]) -> [Int] {
        var oldDigits = digits
        for i in (0..<oldDigits.count).reversed() {
            if oldDigits[i] < 9 {
                oldDigits[i] += 1
                break
            } else {
                oldDigits[i] = 0
                if (i == 0) {
                    oldDigits.insert(1, at:0)
                }
            }
        }
        return oldDigits
    }
}

解析：

反转数组，逐个增加一。如果当前值是9，加一后，设置当前值为0，并记录增长位为一；如果不是9，直接加一。

移动零

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

示例:

输入: [0,1,0,3,12]
输出: [1,3,12,0,0]

说明:

  1. 必须在原数组上操作，不能拷贝额外的数组。
  2. 尽量减少操作次数。

野猫君实现：

耗时 20ms，战胜100%的 Swift 提交记录

class Solution {
    func moveZeroes(_ nums: inout [Int]) {
        if nums.count <= 1 { return }

        var indicator = 0
        for i in 0..<nums.count {
            if nums[i] != 0 {
                nums[indicator] = nums[i]
                indicator += 1
            }
        }
        for i in indicator..<nums.count {
            nums[i] = 0
        }
    }
}

解析：

创建变量indicator记录不为0的位置，从头开始遍历，遇到不为0的数，把它插入indicator位置。最后遍历indicator到末尾，全部设为0.

有效的数独

判断一个 9x9 的数独是否有效。只需要根据以下规则，验证已经填入的数字是否有效即可。

  1. 数字 1-9 在每一行只能出现一次。
  2. 数字 1-9 在每一列只能出现一次。
  3. 数字 1-9 在每一个以粗实线分隔的 3x3 宫内只能出现一次。

上图是一个部分填充的有效的数独。

数独部分空格内已填入了数字，空白格用 '.' 表示。

示例 1:

输入:
[
  ["5","3",".",".","7",".",".",".","."],
  ["6",".",".","1","9","5",".",".","."],
  [".","9","8",".",".",".",".","6","."],
  ["8",".",".",".","6",".",".",".","3"],
  ["4",".",".","8",".","3",".",".","1"],
  ["7",".",".",".","2",".",".",".","6"],
  [".","6",".",".",".",".","2","8","."],
  [".",".",".","4","1","9",".",".","5"],
  [".",".",".",".","8",".",".","7","9"]
]
输出: true

示例 2:

输入:
[
  ["8","3",".",".","7",".",".",".","."],
  ["6",".",".","1","9","5",".",".","."],
  [".","9","8",".",".",".",".","6","."],
  ["8",".",".",".","6",".",".",".","3"],
  ["4",".",".","8",".","3",".",".","1"],
  ["7",".",".",".","2",".",".",".","6"],
  [".","6",".",".",".",".","2","8","."],
  [".",".",".","4","1","9",".",".","5"],
  [".",".",".",".","8",".",".","7","9"]
]
输出: false
解释: 除了第一行的第一个数字从 5 改为 8 以外，空格内其他数字均与 示例1 相同。
     但由于位于左上角的 3x3 宫内有两个 8 存在, 因此这个数独是无效的。

说明:

  • 一个有效的数独（部分已被填充）不一定是可解的。
  • 只需要根据以上规则，验证已经填入的数字是否有效即可。
  • 给定数独序列只包含数字 1-9 和字符 '.' 。
  • 给定数独永远是 9x9 形式的。

class Solution {
    func isValidSudoku(_ board: [[Character]]) -> Bool {
        var rowSet = Set<Character>()
        var columnSet = Set<Character>()
        for i in 0..<9 {
            rowSet.removeAll()
            columnSet.removeAll()
            for j in 0..<9 {
                // row check
                if board[i][j] != "." {
                    if rowSet.contains(board[i][j]) {
                        return false
                    }
                    rowSet.insert(board[i][j])
                }

                // column check
                if board[j][i] != "." {
                    if columnSet.contains(board[j][i]) {
                        return false
                    }
                    columnSet.insert(board[j][i])
                }

                // block check
                if i % 3 == 0 && j % 3 == 0 {
                    if !checkBlock(board, row: i, column: j) {
                        return false
                    }
                }
            }
        }
        return true
    }

    func checkBlock(_ board: [[Character]], row: Int, column: Int) -> Bool {
        var blockSet = Set<Character>()
        for i in row..<(row + 3) {
            for j in column..<(column + 3) {
                if board[i][j] != "." {
                    if blockSet.contains(board[i][j]) {
                        return false
                    }
                    blockSet.insert(board[i][j])
                }
            }
        }
        return true
    }
}

解析：

利用Set的内容具有唯一性，逻辑只关注3点：

  1. 每一行是否符合要求（数字 1 - 9 中每个数只出现一次）
  2. 每一列是否符合要求
  3. 每一个黑色粗线划分的 3x3 区域是否符合要求

旋转图像

给定一个 n × n 的二维矩阵表示一个图像。

将图像顺时针旋转 90 度。

说明：

你必须在原地旋转图像，这意味着你需要直接修改输入的二维矩阵。请不要使用另一个矩阵来旋转图像。

  [1,2,3],

class Solution {
    func rotate(_ matrix: inout [[Int]]) {
        let n = matrix[0].count 
        var temp: Int
        for i in 0..<n {
            for j in (i + 1)..<n {
                temp = matrix[i][j]
                matrix[i][j] = matrix[j][i]
                matrix[j][i] = temp
            }
        }

        for i in 0..<n {
            for j in 0..<(n / 2) {
                temp = matrix[i][j]
                matrix[i][j] = matrix[i][n-1-j]
                matrix[i][n-1-j] = temp
            }
        }
    }
}
